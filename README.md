# Amaara Quiz

## Tech Stack

### Frontend
- React with TypeScript
- Tailwind CSS
- JestJs 
- React Router 

## Backend
- Node
- Express
- MongoDB
- jwt authentication

## Functionalities implemented

- Optional Login with jwt authentication
- Take Quiz
- Dark mode/Light mode toggle
- Check Answers
- Personal Scoreboard
- LeaderBoard 


Live Demo: [Amaara Quiz](https://amaara-quiz.netlify.app/)
